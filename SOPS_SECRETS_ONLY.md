## How to Edit

```bash
# Edit the secrets file
sops secrets/common.yaml

# In the editor, DELETE all configuration blocks
# Keep ONLY passwords, keys, and tokens
# Save and exit
```

## Verification

After editing, verify:
```bash
# Check rust-web-server only has secrets
sops -d secrets/common.yaml | yq '.rust-web-server'
# Should show ONLY: db-password and oauth-client-secret

# Check openldap only has password
sops -d secrets/common.yaml | yq '.openldap'  
# Should show ONLY: admin-password

# Check restic has individual secrets
sops -d secrets/common.yaml | yq '.restic'
# Should show: pass, aws-access-key-id, aws-secret-access-key (NOT the s3 config block)
```

## Remember the Rule

**SOPS = SECRETS ONLY**
- Passwords ✓
- Private keys ✓
- API tokens ✓
- Secret tokens ✓

**TEMPLATES = CONFIGURATION**
- URLs ✗ (goes in template)
- Client IDs ✗ (goes in template)
- Usernames ✗ (goes in template)
- Endpoints ✗ (goes in template)
- Any non-secret config ✗ (goes in template)
